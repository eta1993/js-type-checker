# js-type-checker

JS native types checker using `Object.prototype.toString.call`

## Checkers

- isArray
- isBoolean
- isDate
- isEmpty
  - Checks for empty objects, array, and falsy values 
- isFunction
- isNull
- isNumber
- isObject
- isString
- isUndefined

## Helpers
- inArray(needle, haystack)
- isObjectEmpty
- isTruthy
